# Support

## How to get help

Please use the right channel for your question:

| Type | Where |
|---|---|
| 🐛 Bug report | [Open an issue](https://github.com/Tradeline-hov/Tradeline-hov/issues/new?template=bug_report.md) |
| 💡 Feature request | [Open an issue](https://github.com/Tradeline-hov/Tradeline-hov/issues/new?template=feature_request.md) |
| ❓ General question | [Start a discussion](https://github.com/Tradeline-hov/Tradeline-hov/discussions) |
| 🔒 Security vulnerability | [Private advisory](https://github.com/Tradeline-hov/Tradeline-hov/security/advisories/new) — see [SECURITY.md](./SECURITY.md) |
| 📖 Documentation gap | [Open an issue](https://github.com/Tradeline-hov/Tradeline-hov/issues/new) with label `documentation` |

## Before opening an issue

1. Search [existing issues](https://github.com/Tradeline-hov/Tradeline-hov/issues) — your question may already be answered
2. Check the [README](./README.md) setup guide
3. Check the [CHANGELOG](./CHANGELOG.md) for recent changes that may affect your version

## Response times

This is an open-source project maintained on a best-effort basis.

| Channel | Expected response |
|---|---|
| Security advisories | 24–48 hours |
| Bug reports | 3–5 business days |
| Feature requests | Best effort |
| Discussions | Best effort |
